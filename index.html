Student Database System

A Student Database Management System (SDMS) is a software application designed to efficiently manage and organize student information such as names, roll numbers, courses, grades, attendance, and contact details. It typically includes features like secure login, the ability to add, edit, delete, and view student records, and search or filter functionality to easily access data. The system enhances administrative efficiency by reducing paperwork and enabling quick, centralized access to academic information. Built using technologies like HTML, CSS, JavaScript for the frontend and optionally PHP, Python, or Node.js for the backend with a database like MySQL or Firebase, it can be hosted online or locally, depending on whether dynamic data handling is needed.

Multiobject detection using AI for Visionless people

This project aims to develop a portable smart assistive system for visually impaired individuals, utilizing AI-based object detection techniques to recognize multiple objects in real-time and deliver audio feedback for enhanced spatial awareness. Using models like YOLOv5 (You Only Look Once), the system captures live video through a camera module, processes the feed to detect objects (e.g., people, vehicles, obstacles), converts the identified objects into textual data, and then into speech using a Text-to-Speech engine (TTS). The auditory output is delivered via an earphone or speaker, allowing the user to navigate safely and independently.
